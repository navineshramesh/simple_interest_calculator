# Contributing Guidelines

Thank you for considering contributing to this project!

## How to Contribute

1. Fork the repository.
2. Clone your fork locally.
3. Create a branch for your changes (`git checkout -b feature-name`).
4. Make your changes.
5. Commit your changes with a descriptive message (`git commit -m "Add feature"`).
6. Push your branch to your fork (`git push origin feature-name`).
7. Open a pull request to the main repository.

## Reporting Issues
If you find a bug or have a suggestion, please open an issue in the repository.
